# Elegy Notify

Notification helpers for local toast delivery and outbound webhook calls.

## Capabilities

- `notify-toast`: Display a local toast notification with a title and body. Windows-only.
- `notify-webhook`: Send an outbound HTTP POST webhook with an optional JSON or text payload.
