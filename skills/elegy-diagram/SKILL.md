# Elegy Semantic Diagram Tools

Semantic diagram capabilities for creating, patching, narrating, and rendering structured visual diagrams.

## Capabilities

- `diagram-create`: Initialize an empty semantic diagram of a given type (e.g., concept, architecture, mindmap).
- `diagram-patch`: Surgically add or remove nodes and edges from an existing diagram.
- `diagram-narrate`: Read a semantic diagram and produce a textual human-readable summary of nodes and relations.
- `diagram-render`: Transform a semantic diagram into a visual format such as Mermaid.js.
