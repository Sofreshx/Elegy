# Elegy Skill Authoring

Doctrine for drafting, auditing, and fixing Elegy skills.

## Capabilities

- `skill-audit-checklist`: Walk the objective audit checks against a SKILL.md body or fixture.
- `skill-anti-pattern-lookup`: Look up the canonical anti-pattern catalogue by id and return the pattern, the failure mode, and the replacement.
- `skill-template-emit`: Emit the canonical SKILL.md skeleton from assets/skill-template.md with placeholders preserved.
