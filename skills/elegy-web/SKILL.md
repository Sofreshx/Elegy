# Elegy Web Automation

Bounded HTTP automation for safe fetch and reachability checks over http and https.

## Capabilities

- `web-fetch`: Issue a bounded HTTP request over http or https and return status, headers, text, parsed JSON, and optional extracted JSON value.
- `web-ping`: Perform a bounded reachability check for an http or https URL.
