# Elegy Skill Registry

Registry-first discovery, resolution, and validation over governed skill definitions.

## Capabilities

- `skills-registry-search`: Search the governed skill registry by task or keyword and return ranked matches with context-budget hints.
- `skills-registry-resolve`: Resolve the best matching skill and capability for a task without loading every full skill definition into context.
- `skills-registry-validate`: Validate one skill file or a directory of skill files against the governed skill format plus registry invariants.
