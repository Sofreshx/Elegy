# Elegy Mermaid Projection

Deterministic Mermaid projection tooling over governed canonical workflow inputs.

## Capabilities

- `mermaid-render`: Render a governed workflow JSON input to Mermaid flowchart text.
- `mermaid-reverse`: Reverse Mermaid flowchart TD text into a bounded workflow-graph projection report.
- `mermaid-narrate`: Narrate Mermaid text or governed workflow JSON as a concise derived report.
