# Elegy Skill Router

Search and discover profile-filtered Elegy capabilities on demand through the host-facing agent surface.

## Capabilities

- `router-skill-search`: Find relevant profile-filtered skills and capabilities by keyword or task description.
- `router-skill-describe`: Get detail-level profile-filtered discovery for a specific skill or alias.
- `router-skill-list`: List profile-filtered discoverable skills with compact index-level metadata.
