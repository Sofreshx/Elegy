# Elegy Data Tools

Structured data helpers for format conversion, JSON extraction, and JSON Schema validation.

## Capabilities

- `data-convert`: Convert structured data between json, yaml, toml, and csv.
- `data-extract`: Extract a JSON value using either JSON Pointer or a simple dotted path expression.
- `data-validate`: Validate input JSON against a supplied JSON Schema and return a structured validity report.
