# Elegy-skills agents overlay

This folder is a contributor pointer shell only.

It is not an agent implementation center, orchestration center, or release surface.

Use these locations instead:

- `contracts/` and `governance/` for canonical skill authority.
- `rust/` for reusable executable behavior.
- downstream consuming repos for host-specific agent registration, orchestration, auth, and runtime policy.

This overlay does not reopen a shared in-repo agent package-family story.
