# Elegy-configuration docs helper lane

This folder is the docs helper lane for the `src/Elegy-configuration` wrapper surface.

It does not become a canonical docs center, authority layer, implementation center, or release surface.

Use it to route contributors from the wrapper entrypoint to the owned documentation locations:

- `docs/architecture/elegy-configuration-v1.md` for the current configuration boundary.
- `docs/distribution.md` for release and wrapper install posture.
- `docs/agent-integration.md` for portable package posture.
- `contracts/`, `governance/`, and `rust/` for canonical ownership.
