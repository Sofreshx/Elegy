# `elegy-accounts` — distribution

## Shape

- Portable authority: `.elegy-plugin/plugin.json`
- Binary package: `plugins/accounts/crates/elegy-account-cli`
- Release archive: `elegy-accounts-plugin-<target>.zip`
- Codex projection: generated by `elegy-plugin-packaging`; `.app.json` is not used
- Local data: `%LOCALAPPDATA%\Elegy\Accounts`

The archive includes the broker binary, governed skill, MCP descriptor,
Account Center static assets, and Brave extension assets. The executable starts
on demand; no system-wide service or administrator installation is required.

## Install and verify

```powershell
elegy-plugin-packaging install --archive elegy-accounts-plugin-x86_64-pc-windows-msvc.zip
elegy-accounts status --json
elegy-accounts open
```

Codex consumers install the generated Elegy marketplace projection. Holon
consumes the same binary through its generic `elegy.accounts` compatibility
package; it does not receive a separate credential store.

## Development validation

```powershell
cargo test -p elegy-accountd
cargo test -p elegy-accounts
cargo run -p elegy-tooling --bin elegy-plugin-packaging -- verify --plugin plugins/accounts
```
