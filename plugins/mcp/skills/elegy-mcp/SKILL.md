# Elegy MCP Descriptor Tools

Governed descriptor authoring and analysis for MCP server descriptors.

## Capabilities

- `mcp-author-descriptor`: Create a governed MCP server descriptor from explicit tool names and descriptions.
- `mcp-analyze-descriptor`: Analyze a governed MCP descriptor for tool schema viability and extracted triggers.
